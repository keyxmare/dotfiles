# Module — i18n

- **Backend** : `translations/messages.fr.yaml`, `translations/messages.en.yaml`.
- **Frontend Nuxt** : module `@nuxtjs/i18n` dans `nuxt.config.ts`, fichiers `locales/fr.json`, `locales/en.json`.
- **Frontend Vue** : `vue-i18n` dans `main.ts`, fichiers de traduction JSON par locale.
- Locale par défaut : `fr`. Locales supportées : `fr`, `en`.
